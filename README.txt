------------Introduction------------
J'ai réalisé un jeu de puzzle qui s'appelle LiquidSort. J'ai programmé ce
jeu avec le langage Javascript, Html5 et l'IDE Visual Studio Code. Pour les graphismes
je les ai réalisé moi-même avec Inkscape.


----------règles du jeu---------------
Le but du jeu est de remplir un maximum de bouteille. Pour cela il faut que la bouteille
soit pleine et que le liquide dans la bouteille soit d'une couleur unie.

Attention on ne peut pas verser le liquide dans une bouteille pleine ou dans une 
bouteille qui a un liquide au dessus de la pile d'une couleur différente par rapport au 
liquide qui doit être versé.


--------Déroulement du jeu---------------
Quand on lance le projet, on arrive directement sur l'écran de jeu. 

Pour verser le liquide d'une bouteille dans une autre, il suffit de cliquer avec la 
souris d'abord sur la bouteille où on veut prendre le liquide (bouteille courante), 
et ensuite cliquer sur la bouteille où on veut verser le liquide(bouteille cible).

Si on ne peut pas verser le liquide dans la bouteille cible, alors les bouteilles ce
déselectionne.

Une fois qu'on a rempli le maximum de bouteilles qu'on peut remplir, on passe 
automatiquement au niveau suivant. Plus le nombre de niveau augmente, plus le 
nombre de bouteille à remplir augmente.
